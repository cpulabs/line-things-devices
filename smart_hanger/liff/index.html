LINE Things smart hanger

LINE Thingsを利用して、濡れた服が乾くと、自動的にLINEのメッセージで通知をします。 また、乾くまでの時間を予測し、LINEのLIFF上から知ることができます。

はじめの設定

使用する前に、Settingsタブを開き、Read and set parameterボタンを押し、服をかけていないときの室内の気温・湿度をデバイスに記憶します。 この作業は新しい環境で使用する前にかならず行ってください。なお、SW2を押しながら、デバイスの電源を入れても設定を行うことが可能です。

使い方

LIFFのMainタブで、Startボタンを押すか、デバイスのSW2ボタンを押すことで計測が開始されます。服が乾くとLINEに通知がされますが、このときにLIFFを開いていると、 LINEのメッセージではなく、代わりにLIFF上でステータスが表示されます。現在の状態を確認したい場合は、Read Statusボタンを押すことでデバイスから情報を読み、表示することが可能です。 おかしくなったら、SettingsタブのResetボタンを押すことでリセットすることが可能です。

最大接続数について

SW3はリセットスイッチです。このデバイスはBluetooth LE multi connectionに対応していて、同時にの最大数3台の端末と接続可能です。 3台以上接続している場合は、新規に接続されません。デバイスのリセットボタンSW3ボタンを押してリセットをかけると、現在接続中の端末のコネクションを 強制的に切断することができるので、リセット後に接続を行ってください。
